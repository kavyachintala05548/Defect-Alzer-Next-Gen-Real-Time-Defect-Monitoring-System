# Defect-Alzer-Next-Gen-Real-Time-Defect-Monitoring-System
Defect Alzer is a real-time AI-based defect detection system for manufacturing industries, built using YOLOv8, OpenCV, and Streamlit. It detects product defects, visualizes results with bounding boxes, and generates CSV reports. Designed as a scalable and cost-effective quality control solution.
